# 🎓 AcadVault

AcadVault is a simple Student Portfolio Management System built using React and Firebase.  
It allows staff to manage student records in a clean and organized way.

---

## 🚀 Features

- 🔐 Login system (Firebase Authentication)
- 📊 Dashboard to view all students
- ➕ Add new student details
- 🔍 Search students by name or department
- 🏷️ Department-based filtering
- 🚪 Logout functionality

---

## 🛠️ Tech Stack

- Frontend: React.js
- Backend (BaaS): Firebase
- Database: Firestore
- Authentication: Firebase Auth
- Styling: CSS
